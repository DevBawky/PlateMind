# PlateMind

PlateMind는 야구의 투타 심리전을 구종, 코스, 볼카운트, 결과 확률로 분해해 보여주는 Electron 기반 MVP입니다. 현재 버전은 포트폴리오 제출과 기획 검증을 위한 샘플 데이터로 동작하며, 실제 pitch-by-pitch 데이터와 연결할 수 있도록 구조를 나누어 설계했습니다.

## PlateMind 개요

PlateMind는 투수가 어떤 구종을 어느 구역에 던질 때 타자에게 압박을 줄 수 있는지, 반대로 어떤 선택이 출루나 장타 위험으로 이어질 수 있는지를 시각적으로 확인하는 분석 도구입니다.

화면 중앙의 투구 분포 분석은 Baseball Savant의 pitch breakdown처럼 구종별 투구 위치를 점으로 표현하고, 선택한 구역에 대한 투구 가능성, 예상 타율, 안타 확률, 홈런 확률, 압박 지수, 위험 지수를 함께 보여줍니다.

## 제작 의도

이 프로젝트는 단순한 야구 기록 조회가 아니라, 게임기획 관점에서 “선택과 대응”을 수치화하는 실험입니다.

야구의 한 타석은 투수와 타자가 서로의 성향, 카운트, 이전 선택을 읽고 다음 행동을 고르는 턴제 전투처럼 볼 수 있습니다. PlateMind는 이 구조를 데이터 UI와 간단한 시뮬레이션으로 풀어내어, 스포츠 장르의 전략성과 밸런싱 사고를 보여주는 것을 목표로 했습니다.

## 핵심 기능

- 투수의 구종별 코스 분포 시각화
- 구역 클릭 기반 상세 분석
- 투구 가능성, 예상 타율, 안타 확률, 홈런 확률 표시
- 압박 지수와 위험 지수 기반 추천 구역 도출
- 0-0부터 타석 종료까지 진행되는 타석 시뮬레이션
- 10,000회 반복 결과 기반 출루, 아웃, 안타, 홈런, 볼넷, 삼진 확률 계산
- 대표 타석 로그 제공
- 반응형 UI 구성

## 주요 화면 구성

- 상단 헤더: 앱명, 투수 vs 타자, 샘플 수와 시뮬레이션 횟수
- 좌측 정보 패널: 투수/타자 정보, 구종 구성, 추천 구역, 밸런스 코멘트
- 중앙 투구 분포 분석: 구종별 투구 위치와 구역 선택 UI
- 우측 분석 패널: 선택 구역 상세 분석, 타석 시뮬레이션 결과
- 하단 안내: 샘플 데이터 기반 MVP 안내

## 데이터 안내

현재 데이터는 실제 선수 기록이 아닌 기획 검증용 샘플 데이터입니다.

샘플 데이터는 다음 기준으로 구성되어 있습니다.

- 투수/타자 기본 정보
- 투수별 구종 구성
- 볼카운트
- 구종
- 9개 구역
- 투구 가능성
- 예상 타율
- 안타/홈런/스윙/헛스윙 확률
- 압박 지수
- 위험 지수

추후 실제 pitch-by-pitch 데이터와 연결할 경우, 현재의 `count + pitchType + zoneId` 구조를 유지하면서 데이터 소스만 교체할 수 있도록 설계했습니다.

## 포트폴리오 포인트

- 야구 장르의 투타 심리전을 시스템으로 해석
- 스포츠 데이터를 게임기획 지표로 재구성
- 압박 지수와 위험 지수라는 밸런싱 지표 설계
- 단일 결과 예측이 아니라 선택 구조와 확률 흐름을 보여주는 UI 구성
- Electron 앱 형태로 실행 가능한 MVP 구현
- TypeScript 기반 도메인 모델 분리

## 향후 개선 방향

- 실제 MLB 또는 KBO pitch-by-pitch 데이터 연동
- 투수별/타자별 성향 파라미터 추가
- 구종별 릴리스, 구속, 무브먼트 데이터 반영
- 카운트별 추천 로직 고도화
- 시뮬레이션 결과의 재현 가능한 seed 옵션 추가
- 선수 검색과 매치업 비교 기능 추가
- 포트폴리오 발표용 화면 캡처와 기획서 연동
