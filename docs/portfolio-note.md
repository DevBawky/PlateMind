# 포트폴리오 노트

## 프로젝트가 보여주는 역량

PlateMind는 야구 데이터 분석 화면이면서 동시에 게임기획 포트폴리오다. 단순히 데이터를 표시하는 것이 아니라, 투수와 타자의 선택 구조를 규칙과 지표로 재구성했다.

## 데이터 분석 역량

샘플 데이터 기반 MVP이지만 데이터 구조는 실제 pitch-by-pitch 데이터로 확장할 수 있도록 설계했다.

- 볼카운트
- 구종
- 구역
- 결과 확률
- 압박 지수
- 위험 지수

이 조합을 통해 단일 기록이 아니라 상황별 선택 구조를 분석할 수 있다.

## 야구 장르 이해도

한 타석의 핵심은 구종 선택, 코스 선택, 볼카운트 변화, 타자의 스윙 판단이다. PlateMind는 이 흐름을 화면과 시뮬레이션으로 표현해 야구 장르의 전략성을 드러낸다.

## 시스템 기획 역량

압박 지수와 위험 지수는 기획자가 밸런싱 관점에서 조정할 수 있는 핵심 파라미터다. 어떤 선택이 안정적인지, 어떤 선택이 위험하지만 보상이 큰지 설명할 수 있게 한다.

## 밸런싱 사고

PlateMind는 추천 구역을 단순히 낮은 피안타율만으로 결정하지 않는다. 투구 가능성, 압박 지수, 위험 지수를 함께 고려한다. 이는 게임에서 리스크와 리턴을 조정하는 방식과 닮아 있다.

## 개발 이해도

Electron, React, TypeScript를 사용해 실행 가능한 MVP를 구현했다. 도메인 모델, 샘플 데이터, 시뮬레이션 서비스, UI 컴포넌트를 분리해 확장 가능성을 확보했다.
