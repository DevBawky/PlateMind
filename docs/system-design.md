# PlateMind 시스템 설계

## 압박 지수

압박 지수는 현재 구종, 코스, 볼카운트가 타자에게 주는 심리적 압박을 수치화한 값이다.

높은 압박 지수는 다음 의미를 가진다.

- 타자가 쉽게 스윙 결정을 내리기 어렵다.
- 헛스윙 또는 불리한 카운트로 이어질 가능성이 높다.
- 투수가 주도권을 잡고 있는 선택지에 가깝다.

## 위험 지수

위험 지수는 해당 투구 선택이 출루, 장타, 제구 실패로 이어질 위험을 수치화한 값이다.

높은 위험 지수는 다음 의미를 가진다.

- 타자가 강하게 대응할 가능성이 있다.
- 안타 또는 홈런으로 이어질 수 있다.
- 볼넷이나 불리한 카운트로 흐를 가능성이 있다.

## 구종/구역/볼카운트 기반 분석 구조

PlateMind의 핵심 데이터는 다음 조합으로 구성된다.

- 볼카운트
- 구종
- 구역

각 조합은 다음 값을 가진다.

- 투구 가능성
- 예상 타율
- 안타 확률
- 홈런 확률
- 스윙 확률
- 헛스윙 확률
- 압박 지수
- 위험 지수

이 구조를 사용하면 특정 구종이나 특정 구역만 보는 것이 아니라, 상황별 선택 구조를 확장할 수 있다.

## 투수와 타자 성향을 수치화하는 방식

현재 MVP는 샘플 데이터 기반으로 동작한다. 투수 성향은 구종 구성과 구역별 투구 가능성으로 표현하고, 타자 성향은 구역별 예상 타율, 스윙 확률, 헛스윙 확률, 장타 위험으로 표현한다.

향후 실제 데이터와 연결하면 다음 값으로 확장할 수 있다.

- 투수의 구종 사용률
- 타자의 구종별 대응 성적
- 카운트별 스윙 경향
- 존별 타구 품질
- 좌우 매치업 성향
